# RBAC + UBAC MERN Application

##  Features

- JWT Authentication
- Role-Based Access Control (RBAC)
- User-Based Access Control (UBAC)
- Dynamic Permission System
- Admin Panel for Role & User Permissions
- Scalable MVC + Service Layer Architecture

---

## 🛠 Tech Stack

Backend:
- Node.js
- Express.js
- TypeScript
- MongoDB (Mongoose)

Frontend:
- React.js
- Context API
- Axios

---
